## Performance

### Optimization Techniques
1. Lazy loading (modal only loads when opened)
2. Code splitting (markdown renderer)
3. Memoization (file tree rendering)
4. Debouncing (copy button feedback)
5. Virtual scrolling (for large file lists)

### Bundle Size Impact
- react-markdown: ~50KB
- react-syntax-highlighter: ~100KB
- Total added: ~150KB (gzipped: ~40KB)
